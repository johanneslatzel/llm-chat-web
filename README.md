A web tool suite for the `llm-chat` ecosystem. Supports web search (Tavily, ExaAI, DuckDuckGo) and URL fetching with content extraction.

## Prerequisites

- Node.js >= 18

## Installation

```bash
npm install @johannes.latzel/llm-chat-web
```
